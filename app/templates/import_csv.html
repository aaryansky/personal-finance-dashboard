Import Transactions from CSV

Upload a CSV file with your bank transactions to import them automatically.

Important: Your CSV file must have the following columns in this exact order:

  1. Date (format: YYYY-MM-DD)
  2. Amount (a positive number)
  3. Type (either 'income' or 'expense')
  4. Category (e.g., Groceries, Salary)
  5. Description (optional)
